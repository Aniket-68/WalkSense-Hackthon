# WalkSense – Requirements Document

## 1. Project Overview

WalkSense is a real-time AI vision–language assistive system designed to enhance environmental awareness and safety for visually impaired individuals using camera-based perception and voice interaction.

## 2. Problem Statement

Visually impaired individuals lack real-time environmental awareness and contextual guidance required for safe and independent mobility.

## 3. Objectives

- Provide real-time obstacle and hazard detection
- Deliver contextual scene understanding using multimodal AI
- Enable voice-based hands-free interaction
- Support multilingual accessibility
- Enable offline-capable edge deployment

## 4. Functional Requirements

- Capture real-time video using camera input
- Detect obstacles and hazards using object detection models
- Provide contextual scene descriptions
- Read environmental text using OCR
- Accept voice queries from the user
- Deliver spoken responses and alerts
- Prioritize safety alerts over all outputs

## 5. Non-Functional Requirements

- Low-latency real-time processing
- Offline/edge execution capability
- Scalable modular architecture
- High reliability for safety alerts
- Multilingual voice output support

## 6. Target Users

- Visually impaired individuals
- Assistive mobility device manufacturers
- Accessibility technology providers
